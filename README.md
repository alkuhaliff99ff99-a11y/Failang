🚀 Failang (FSL) — لغة فاي

Failang هي لغة برمجة عربية حديثة قيد التطوير، تهدف إلى تقديم تجربة برمجية سهلة وواضحة مع دعم اللغة العربية والإنجليزية، وبناء بيئة تطوير حديثة للمبرمجين.

🌟 الرؤية

إنشاء لغة برمجة تجمع بين:

- سهولة الاستخدام مثل Python.
- الأداء والكفاءة في تنفيذ البرامج.
- كتابة أوامر برمجية عربية واضحة.
- تصميم حديث قابل للتوسع والتطوير.

✨ المميزات الحالية

- ✅ دعم الكلمات المفتاحية العربية.
- ✅ العمليات الحسابية.
- ✅ الشروط والتحكم بالتدفق.
- ✅ الدوال البرمجية.
- ✅ المصفوفات الديناميكية.
- ✅ بيئة تشغيل للبرامج.
- ✅ تصميم قابل لإضافة ميزات مستقبلية.

📝 مثال برمجي

دالة فحص(قائمة):
    إذا طول(قائمة) > 2:
        ارجع "القائمة طويلة"

    ارجع "القائمة قصيرة"


متغير النتيجة = فحص([1, 2, 3])

اطبع النتيجة

الناتج:

القائمة طويلة

⚙️ تشغيل برنامج Failang

بعد تثبيت بيئة Failang:

fsl البرنامج.fsl

مثال:

fsl hello.fsl

📂 امتداد الملفات

تستخدم ملفات Failang الامتداد:

.fsl

مثال:

hello.fsl

🏗️ حالة المشروع

🚧 Failang حاليًا في مرحلة التطوير النشط.

يتم العمل على:

- تطوير المترجم.
- تحسين بيئة التشغيل.
- إضافة مكتبات قياسية.
- بناء أدوات التطوير.

🎯 الهدف المستقبلي

تسعى Failang إلى أن تصبح لغة برمجة حديثة تدعم:

- تطبيقات سطح المكتب.
- تطوير الأدوات والبرامج.
- البرمجة التعليمية.
- الأنظمة عالية الأداء.
