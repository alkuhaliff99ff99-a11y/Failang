# 🚀 Failang (FSL) — لغة فاي
**Failang** هي لغة برمجة عربية/إنجليزية حديثة ومبتكرة، صُممت لتقديم تجربة برمجية مرنة وبسيطة مع الحفاظ على الأداء العالي. تدمج اللغة بسلاسة بين الكلمات المفتاحية باللغتين العربية والإنجليزية، مما يمنح المبرمجين أداة قوية تدعم الإبداع بلغتهم الأم واللغة البرمجية العالمية.
## 🌟 الرؤية والأهداف
تسعى **Failang** إلى تحقيق التوازن بين سهولة الاستخدام (مثل بايثون) وبين الكفاءة والأداء، مع التركيز على بناء بيئة تطوير متكاملة تسهل على المبرمجين كتابة، تنسيق، وإدارة مشاريعهم البرمجية بشكل احترافي.
## ✨ المميزات الحالية
 * **ثنائية لغوية ذكية:** دعم كامل للمرادفات العربية والإنجليزية (مثل اطبع/print).
 * **منسق أكواد آمن (fsl fmt):** أداة مدمجة لترتيب الأكواد وتنسيقها مع حماية النصوص والرموز.
 * **العمليات الحسابية والمنطقية:** معالجة متقدمة ومقارنات نصية طبيعية (مثل يساوي).
 * **بيئة تفاعلية:** دعم وضع (REPL) وأوامر سطرية (CLI) لتسهيل التنفيذ.
## 📝 مثال برمجي (Bilingual Code Example)
```fsl
دالة فحص(قائمة):
    إذا طول(قائمة) > 2:
        ارجع "القائمة طويلة"
    ارجع "القائمة قصيرة"

متغير النتيجة = فحص([1, 2, 3])
اطبع النتيجة  # المخرجات: القائمة طويلة

```
## 🛠️ أدوات الطرفية (CLI Commands)
توفر البيئة مجموعة من الأوامر لإدارة المشاريع:
 * fsl run file.fsl: لتشغيل ملفات اللغة.
 * fsl fmt file.fsl: لتنسيق وترتيب الكود تلقائياً.
 * fsl new ProjectName: لإنشاء مشروع جديد.
 * fsl test: لتشغيل الاختبارات الآلية.
 * fsl build: لتجميع المشروع.
## 🏗️ خارطة الطريق (Roadmap)
المشروع حالياً في مرحلة **التطوير النشط**، حيث يتم العمل على:
 1. توسيع بيئة المفسر (Interpreter) لدعم ميزات أكثر تعقيداً.
 2. بناء مكتبة قياسية (Standard Library) غنية بالوظائف.
 3. تطوير أدوات تصحيح الأخطاء (Debugger).
