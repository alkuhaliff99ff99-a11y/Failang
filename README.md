🚀 Failang (FSL) — لغة فاي

Failang (FSL) هي لغة برمجة حديثة ثنائية اللغة (العربية والإنجليزية)، تهدف إلى توفير تجربة برمجية سهلة، واضحة، وعالية الأداء. صُممت لتسمح للمطورين بالبرمجة باستخدام الكلمات المفتاحية العربية أو الإنجليزية ضمن لغة واحدة، مع بنية قابلة للتوسع نحو مترجم وأدوات تطوير متكاملة.

---

🌟 الرؤية

تسعى Failang إلى الجمع بين:

- سهولة القراءة والكتابة.
- الأداء والكفاءة.
- دعم العربية والإنجليزية بصورة طبيعية.
- أدوات تطوير حديثة للمبرمجين.
- تصميم قابل للتوسع إلى Compiler وVirtual Machine وNative Backend.

---

✨ المميزات الحالية

- ✅ دعم الكلمات المفتاحية العربية.
- ✅ بداية دعم الكلمات المفتاحية الإنجليزية والمرادفات (قيد التوسع).
- ✅ تعريف واستدعاء الدوال بالأسلوب التقليدي والحر (Free Function Syntax).

- ✅ الشروط والتدفق المنطقي.
- ✅ المقارنات النصية باستخدام يساوي.

  - "build"

# 🚀 Failang (FSL) — لغة فاي

**Failang** هي لغة برمجة عربية/إنجليزية حديثة ومبتكرة، صُممت لتقديم تجربة برمجية مرنة وبسيطة مع الحفاظ على الأداء العالي. تدمج اللغة بسلاسة بين الكلمات المفتاحية باللغتين العربية والإنجليزية لتمنح المبرمجين والمهتمين بالبرمجة التعليمية والتطبيقية أداة قوية تدعم الإبداع بلغتهم الأم واللغة البرمجية العالمية.

---

## 🌟 الرؤية والأهداف

تسعى **Failang** إلى تحقيق التوازن المثالي بين:
- **سهولة الاستخدام:** كتابة أكواد واضحة ومقروءة شبيهة بلغة Python.
- **ثنائية اللغة الكاملة:** دعم مرن للمرادفات العربية والإنجليزية (مثل `اطبع`/`اكتب`/`print`).
- **الكفاءة والأداء:** تصميم قوي ومستقر لبيئة التشغيل وسرعة المعالجة.
- **أدوات تطوير متكاملة:** بيئة سطر أوامر حديثة تدعم التطوير والنسخ الاحتياطي وتنسيق الكود تلقائيًا.

---

## ✨ المميزات الحالية

- [x] **ثنائية لغوية ذكية:** دعم مرادفات الكلمات المفتاحية باللغتين العربية والإنجليزية بالتوازي.
- [x] **تنسيق ذكي وآمن للكود (`fsl fmt`):** أداة تنسيق مدمجة تضمن بقاء نصوصك داخل علامات الاقتباس آمنة مع ترتيب المسافات والمحاذاة تلقائيًا.
- [x] **دعم الدوال الحرة والكتل:** مرونة عالية في كتابة الدوال والشروط والتحكم بالتدفق البرمجي.
- [x] **العمليات الحسابية والمنطقية:** معالجة متقدمة ومقارنات نصية طبيعية مثل `يساوي` و `مساوي`.
- [x] **بيئة تفاعلية (REPL) وأوامر سطرية مدمجة (CLI).**

## 🛠️ أدوات الطرفية (CLI Commands)
توفر البيئة مجموعة من الأوامر لإدارة المشاريع:
 * fsl run file.fsl: لتشغيل ملفات اللغة.
