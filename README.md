# 🚀 Failang (FSL) — لغة فاي

**Failang** هي لغة برمجة عربية/إنجليزية حديثة ومبتكرة، صُممت لتقديم تجربة برمجية مرنة وبسيطة مع الحفاظ على الأداء العالي. تدمج اللغة بسلاسة بين الكلمات المفتاحية باللغتين العربية والإنجليزية لتمنح المبرمجين أداة قوية تدعم الإبداع بلغتهم الأم واللغة البرمجية العالمية.

---

```fsl
دالة فحص(قائمة):
    إذا طول(قائمة) > 2:
        ارجع "القائمة طويلة"
    ارجع "القائمة قصيرة"

متغير س = 10 + 20
متغير النتيجة = فحص([1, 2, 3])
اطبع النتيجة  # المخرجات: القائمة طويلة
