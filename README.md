# 🚀 Failang (FSL) — لغة فاي

<p align="center">
<b>Fast • Safe • Logical</b>
</p>

لغة برمجة عربية/إنجليزية حديثة مبنية بلغة Rust، تهدف إلى الجمع بين سهولة الكتابة وقوة الأداء، مع دعم الكلمات المفتاحية العربية والإنجليزية في نفس المشروع.

---

# ✨ لماذا Failang؟

Failang مصممة لتكون لغة برمجة حديثة:

- 🇸🇦 دعم البرمجة باللغة العربية.
- 🌍 دعم اللغة الإنجليزية.
- ⚡ أداء سريع بفضل Rust.
- 🛡️ بنية آمنة وقابلة للتطوير.
- 🧩 مكتبة قياسية مدمجة.
- 🔄 تصميم قابل للتوسع.
